Are We * Yet is a template site for sharing updates to Mozilla projects.

  • A list of all "are we yet" project sites is kept at https://wiki.mozilla.org/Areweyet
  • The source for this projet can be found at https://github.com/bgrins/areweyet
  • A "burnup chart" tool for Bugzilla can be found at burnup/
